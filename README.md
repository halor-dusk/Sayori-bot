# Sayori bot

Sayori is a discord chat bot made to be my and everyone's budy, she will pretend to 
be Sayori from Doki Doki literature club and will role play some times, she is cute, kind
and very friendly and innocent.


## Features

- Acts like Sayori
- Has chat memory
- Knows who sent which message
- Can see images and gifs
- Has knolowge about internet culture
- Is friendly and inocent
- Can join in calls (still doesn't do anything)


## How to contribute?
Just clone or fork this repository and download the dependences in `requirements.txt` using the following command line: `pip install -r requirements.txt`, feel free to make the changes you want and make a pull request
